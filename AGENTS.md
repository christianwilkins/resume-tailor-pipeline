# AGENTS.md

## Skills
### Available skills
- resume-tailor: Tailor `resume.tex` and application answers from `master-resume.tex`, a job posting file or URL, and the repo's LaTeX workflow. Use for one-page resume targeting, company-specific keyword alignment, application answer drafting, and compile/page-count validation. (file: ./skills/resume-tailor/SKILL.md)

### How to use skills
- Trigger `resume-tailor` whenever the user asks to tailor a resume, generate application answers, or adapt this repo to a specific role.
- Prefer the local prompt in `prompts/tailor-resume.md` when a single instruction file is more convenient than a skill.
